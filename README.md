# 🚀 CodeLearn

**CodeLearn** is an interactive coding practice platform designed to help developers improve their programming skills through real-world challenges, competitions, and progress tracking.

---

## 🌟 Features

- 💻 Real Coding Challenges  
- 📊 Difficulty Levels (Easy, Medium, Hard)  
- 📈 Track Your Progress  
- 🏆 Leaderboard System  
- 🌍 Community Driven Learning  

---

## 🎯 Purpose

This project is built to:
- Improve problem-solving skills  
- Practice frontend development  
- Build a portfolio-ready real-world project  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript  
- Firebase (for authentication & leaderboard)
